# Installation

## Download

```sh
brew install pnpm
git clone git@github.com:ethanneff/playground-react-native.git
cd example
pnpm install
```

## Run

```sh
pnpm ios
```

```sh
pnpm android
```

```sh
pnpm web
```

## Lint

```sh
pnpm lint
```

## Test

```sh
pnpm test
```

## Deploy

```sh
pnpm build
```
